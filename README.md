# Deltamod
Deltamod is an easy to use Deltamod mod patcher built in Electron and other languages (packaged in CLI tools).

[![](https://gamebanana.com/wips/embeddables/94135?type=large)](https://gamebanana.com/wips/94135)

# How to compile (advanced)
## External requirements
### `GM3P` (by techy804)
To use Deltamod from source, you must download the latest GM3P release.<br />
1) Download the latest build of GM3P from your preferred provider (GameJolt, GitHub...)
2) Unzip GM3P in a folder.
3) In the Deltamod root folder, create a new folder named `gm3p`.
4) Move everything from the unzip location of GM3P to this folder.

## Run Deltamod

  <img width="200" alt="nodejs-1-logo-png-transparent" src="https://github.com/user-attachments/assets/a5b53f84-074a-4218-9578-c8fa4cf34683" />
</p>

<br />

1) First of all, you will need the Node.js toolchain since Deltamod is built in Electron. Download it [here](https://nodejs.org/en/download) and install it.
2) Run ``node --version`` in the Command Prompt and check that it is installed correctly. The output should be something along the lines of ``v.1.2.3`` (any version after ``v.22.15.0`` works).
3) Now that you've done that, cd to your Deltamod source code and type ``npm i``. This should have created a folder named ``node_modules`` in the source code folder.
4) After the command finishes, you can now type ``npm test`` to open Deltamod, or use the shortcut file named ``Run.cmd``.

In order to enable the Chromium DevTools, please create an enviroment variable named ``DELTAMOD_ENV`` and set it to ``dev``. The `Run.cmd` file automatically sets this variable for the session.

## Notes
- **Deltamod has the ability to download Deltarune from the itch.io page.**<br />
itch.io servers require a CSRF token which Deltamod _can_ indeed obtain. However, we've chose to not include the CSRF token generation logic in the public source code.<br /><br />
As for a legal standpoint, this is a bit of a gray area. Deltamod itself doesn't contain any copyrighted secrets or encryption keys, even in the `ItchKeys.js` file. It only uses tokens generated by the server and extracts them for use in Deltamod. However, we've still chose to remove it from the GitHub.<br /><br />
_Note: If you have access to the `ItchKeys.js` file you can put it in the `node` folder and the Deltarune download function will be rendered available._

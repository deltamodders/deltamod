Locate the Deltarune files

Welcome, and thanks for downloading Deltamod.

The tool will need to modify the various data.win files in order to add mods to your game. You will now setup Deltamod to point to your Deltarune installation.

Select your DELTARUNE installation folder.




Your installation will be copied to the AppData folder for Deltamod. This will allow us to backup and modify it. You can add more installations later on with the Install Manager.



The BGM playing in the background can also be disabled in the Options.
